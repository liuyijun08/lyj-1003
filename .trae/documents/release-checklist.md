## 三、发布前验证流程

### 步骤 1：静态检查

```bash
# 类型检查
npm run check

# 代码规范
npm run lint
```

### 步骤 2：自动化测试

```bash
# 运行全部单元测试（含变更单完整流程）
npm run test:run
```

### 步骤 3：构建验证

```bash
# 生产构建
npm run build

# 本地预览构建产物
npm run preview
```

### 步骤 4：浏览器手动验收

按上述六大模块的验收清单，在 `npm run dev` 或 `npm run preview` 启动的页面中逐项核对。重点覆盖所有 **P0** 级验收项，P1 和 P2 建议在时间允许时完成。

### 步骤 5：发布通过标准

- [ ] `npm run check` 0 errors
- [ ] `npm run lint` 0 errors
- [ ] `npm run test:run` 全部测试通过
- [ ] `npm run build` 构建成功无报错
- [ ] 浏览器验收清单所有 **P0** 项全部通过
- [ ] 浏览器验收清单 **P1** 项通过率 ≥ 90%
